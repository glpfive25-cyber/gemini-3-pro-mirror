2. 设BD = x，则DC = 14 - x

BD/DC = AB/AC
x/(14-x) = 13/15

15x = 182 - 13x
28x = 182
x = 182/28 = 6.5

【验证】
BD/DC = 6.5/7.5 = 13/15 ✓
